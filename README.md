<h1>Baldi Full Game Api for BepInEx Plugins!</h1>

<p>
    It was designed to make apis for Baldi Full Game conveniently!

    <br>It also used unity interface the BepInEx plugin.

                    </p>
                </li>
            </ul>
        </details>
    </li>
    <li>
        In-game assets look up table

    <li>
